# Цель работы
Реализовать алгоритм Пиявского для поиска глобального экстремума функции
# Алгоритм

- Задать функцию
- Задать вспомогательную функцию
- Расчитать параметры u, p для первого шага
- Расчитать параметры u, p циклично до выполнения условия
